# Avangard Test Task Frontend

## **Getting Started**

### **Требования**
- Node.js (v16 или выше)

### **Установка**

1. Клонировать репозиторий
2. Установить зависимости:
  ```
  npm install
  ```
3. Создайте .env файл в корне проект с переменной VITE_API_URL (API URL Backend)

### **Доступные скрипты**

1. Запуск проекта:
  ```
  npm run dev
  ```
2. Запуск ESLint:
  ```
  npm run lint
  ```
